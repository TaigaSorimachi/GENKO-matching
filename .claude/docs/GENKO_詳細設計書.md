# ブルーカラー人材マッチングプラットフォーム 詳細設計書

## プロジェクト名：**GENKO（ゲンコ）— 現場の声を、力に変える。**

> "現場で働くすべての人に、最速で最適な出会いを届ける"

---

## 1. エグゼクティブサマリー

### 1.1 プロジェクト概要

GENKOは、建設・製造・運輸・物流・警備・介護・清掃など、ブルーカラー産業に特化した次世代型の人材マッチングプラットフォームである。AI駆動のスキル評価、位置情報連動マッチング、60秒応募体験を核とし、「求職者が1分で仕事を見つけ、企業が1日で人材を確保できる」世界を目指す。

### 1.2 ミッション

**「現場を支える人材インフラを、テクノロジーで再定義する」**

ブルーカラー産業は日本のGDPの約30%を支えるにも関わらず、採用プロセスのデジタル化は大きく遅れている。GENKOは、ホワイトカラー向けに設計された既存の採用プラットフォームの構造的課題を解消し、現場労働者のリアルなニーズに寄り添ったマッチング体験を提供する。

### 1.3 市場背景

| 指標 | 数値 |
|------|------|
| 日本のブルーカラー労働者数 | 約1,800万人 |
| 2030年までの製造業人材不足予測（米国Deloitte調査参考） | 210万人 |
| ホワイトカラーからブルーカラーへの転職意欲（XMile調査2025） | 約71%が条件次第で検討 |
| AI代替リスク認識が強い層の現場職志向 | 顕著に高い |
| ブルーカラー産業のDX浸透率 | 他産業比で大幅に低い |

## 2. ターゲットユーザー

### 2.1 求職者ペルソナ

**ペルソナA：「即戦力の現場ベテラン」**
- 年齢：35〜55歳 / 男性
- 職種：大型トラックドライバー、溶接工、電気工事士
- 特徴：資格複数保有、スマホは使うがPC操作は苦手
- ニーズ：条件の良い現場をすぐに見つけたい。面倒な履歴書作成を避けたい
- 課題：既存の求人サイトはUIが複雑で、ホワイトカラー向けの設計に疲弊

**ペルソナB：「業界チェンジャー」**
- 年齢：25〜35歳 / 男女
- 前職：事務職、営業職などホワイトカラー
- 特徴：AI代替不安から現場職に興味。未経験だが体力と意欲あり
- ニーズ：どんな資格が必要か知りたい。現場の雰囲気を事前に把握したい
- 課題：ブルーカラーの職種体系がわからない。情報が断片的

**ペルソナC：「外国人技能労働者」**
- 年齢：20〜40歳 / 男女
- 在留資格：特定技能、技能実習
- 特徴：日本語レベルN3〜N4、母国での実務経験あり
- ニーズ：多言語対応、ビザ要件と連動した求人検索
- 課題：既存システムは日本語前提。資格の国際互換性が無視されている

### 2.2 企業ペルソナ

**企業A：「中小建設会社」**
- 従業員数：10〜50名
- 課題：慢性的な人手不足、採用担当が不在、求人広告のROIが不透明

**企業B：「大手物流企業」**
- 従業員数：500名以上
- 課題：高離職率、シフト充足率の不安定、大量採用の効率化

**企業C：「人材派遣会社」**
- 課題：案件と人材のリアルタイムマッチング、稼働率の最大化

## 3. コア機能設計

### 3.1 求職者向け機能

#### 3.1.1 60秒クイックレジストレーション

従来の履歴書ベースの登録を廃止し、対話型UIで60秒以内に基本登録を完了させる。

```
[登録フロー]
Step 1: 電話番号 → SMS認証（10秒）
Step 2: 名前・年齢・居住エリア（10秒）
Step 3: 興味のある業界をタップ選択（10秒）
        建設 / 製造 / 運輸 / 物流 / 警備 / 介護 / 清掃 / その他
Step 4: 保有資格をAI候補から選択 or 写真撮影で自動読取（15秒）
Step 5: 希望条件（給与・勤務地・シフト）スライダー設定（15秒）
→ 登録完了 → 即座にマッチング結果表示
```

**技術要素：**
- OCR（光学文字認識）による資格証・免許証の自動読取
- 音声入力対応（手が塞がっている現場作業者への配慮）
- AIによる未入力項目の推定補完

#### 3.1.2 AIスキルプロファイリング

履歴書不要の、AI対話型スキル評価システム。

- **AIチャットインタビュー**：5〜10分の自然言語対話で実務経験・スキルレベルを構造化
- **多言語対応**：日本語、英語、ベトナム語、インドネシア語、ミャンマー語、中国語
- **スキルの自動タグ化**：対話内容から「溶接（TIG/MIG）」「フォークリフト（リーチ/カウンター）」等の粒度でスキルタグを自動生成
- **資格の国際互換マッピング**：海外取得資格を日本の対応資格に自動マッピングし、不足分の取得ガイドを提示

#### 3.1.3 ジオマッチング（位置情報連動求人）

```

                             │

                    │  (車/電車/自転車)  │

                             │
              ┌──────────────▼──────────────┐
              │  AIスコアリング                │
              │  ┌──────────────────────────┐│

              │  │ 通勤利便性    × 25%     ││
              │  │ 給与希望合致  × 20%     ││
              │  │ 職場文化適合  × 10%     ││
              │  │ キャリア成長性 × 5%      ││

- 求人の現場住所をGPSベースでマッピングし、実際の通勤時間（車/公共交通機関）を算出
- 「自宅から30分以内」「車通勤OK」等のフィルタをワンタップで適用
- 地図UIで求人を視覚的に探索可能

#### 3.1.4 スキルバッジ & キャリアパスポート

ブロックチェーン技術を活用した、改ざん不可能なスキル証明システム。

- **デジタルバッジ**：取得資格、実務経験年数、企業からの評価を統合したデジタル証明
- **キャリアパスポート**：転職のたびにリセットされない累積型のポートフォリオ
- **スキルツリー表示**：現在の保有スキルと、次のキャリアステップに必要なスキルを視覚化
- **資格取得ナビゲーション**：推奨資格、取得費用、最寄りの試験会場・講習情報を統合表示

#### 3.1.5 現場リアルプレビュー

「入ってみたら想像と違った」を防ぐための情報透明化機能。

- **15秒現場動画**：企業が投稿する現場の日常風景ショートムービー
- **360°VR職場ツアー**：工場・建設現場などをVRで事前体験
- **匿名口コミ**：現役・元従業員によるリアルな職場評価（5段階 + テキスト）
  - 評価軸：安全性 / 人間関係 / 給与満足度 / 体力負荷 / 成長機会
- **「一日体験」予約機能**：正式応募前に半日〜1日の体験勤務を予約

### 3.2 企業向け機能

#### 3.2.1 スマート求人作成エンジン

AIが求人票作成を自動化し、採用担当不在の中小企業でも簡単に求人を公開できる。

```

- 法令遵守チェック（労基法・安衛法に抵触する条件の自動検出）
- 多言語同時翻訳で外国人求職者への自動リーチ

#### 3.2.2 AIタレントスカウト

企業が受動的に応募を待つのではなく、能動的に人材を発掘するシステム。

- **パッシブ候補者発見**：転職意向を明示していないが条件にマッチする登録者をAIが特定
- **スカウトメッセージ自動生成**：候補者のプロフィールに合わせたパーソナライズメッセージ
- **レスポンス予測スコア**：各候補者の返信確率をAIが予測（無駄なスカウトを削減）
- **競合分析ダッシュボード**：同業他社の求人条件との比較分析

- 勤怠管理システム（GPS打刻 / 顔認証打刻）
- 給与前払い機能（働いた分の即日引き出し）

- **ゴールドバッジ**：信頼スコア80以上で取得、優先マッチングの特典

### 4.2 🤝 現場つなぎ — 職人コミュニティ

孤立しがちなブルーカラー労働者のための相互支援コミュニティ。

- **業界別フォーラム**：建設/製造/運輸等のカテゴリ別掲示板
- **先輩に聞ける**：経験豊富な職人に匿名で相談できるメンター機能
- **道具レビュー**：工具・安全装備のクチコミ共有
- **現場あるある**：日常の共感投稿（息抜きコンテンツ）
- **地域ネットワーク**：同じエリアの職人同士のオフライン交流支援

### 4.3 🛡️ セーフティネット — 労働者保護機能

ブルーカラー特有の労働リスクに対応する安全保障機能。

| 機能 | 内容 |
|------|------|
| **賃金未払いアラート** | 給与支払日を超過した場合に自動通知、相談窓口へ接続 |
| **安全違反レポート** | 現場の安全基準違反を匿名で報告できるホットライン |
| **労災サポートナビ** | 労災発生時の手続きガイド、弁護士マッチング |
| **天候アラート** | 建設現場等で危険な気象条件時に自動警告 |
| **ヒートマップ** | 業界別の労働環境リスクスコアを可視化 |
| **外国人労働者SOSライン** | 多言語対応の緊急相談窓口（24時間） |

### 4.4 🧠 AIキャリアコパイロット

求職者一人ひとりに寄り添うAIアドバイザー。

- **キャリアシミュレーション**：「今のスキルで5年後どうなる？」をAIが予測
- **年収最適化提案**：同スキルの全国平均と比較し、年収アップの具体的アクションを提示
- **転職タイミング予測**：市場の需給バランスからベストな転職時期をアドバイス
- **面接コーチング**：AIとの模擬面接で受け答えを練習
- **日本語学習支援**：外国人労働者向けの現場で使う日本語フレーズ学習

### 4.5 ⚡ インスタントペイ — 即日給与アクセス

「働いたら、すぐもらえる」を実現する給与前払いシステム。

- 勤務確認完了後、当日中に稼働分の給与を引き出し可能
- 手数料は業界最低水準（振込1回あたり¥200〜）
- 企業側のキャッシュフローに影響しない立替払いスキーム
- デジタルウォレット連携（PayPay、LINE Pay等）

## 6. セキュリティ & コンプライアンス

### 6.1 セキュリティ設計

| 対策 | 実装 |
|------|------|
| 認証 | OAuth 2.0 + JWT、SMS OTP認証 |
| 暗号化 | 通信: TLS 1.3 / 保存: AES-256 |
| 個人情報保護 | 個人情報の匿名化処理、マスキング表示 |
| アクセス制御 | RBAC (Role-Based Access Control) |
| 脆弱性対策 | OWASP Top 10準拠、定期ペネトレーションテスト |
| ログ管理 | 全操作の監査ログ記録、90日間保持 |

### 6.2 法令遵守

- **個人情報保護法**：プライバシーポリシー明示、同意管理、データポータビリティ
- **職業安定法**：有料職業紹介事業許可の取得（厚労省認可）
- **労働者派遣法**：派遣事業を行う場合の許可取得
- **労働基準法**：求人条件の法令遵守チェック自動化
- **出入国管理法**：在留資格に応じた就労制限の自動フィルタリング
- **特定技能制度対応**：ビザカテゴリに連動した求人マッチング

## 9. ロードマップ

### Phase 1: MVP（0〜6ヶ月）— "まず動く、最小で最速"

| 月 | マイルストーン |
|----|------------|
| M1-M2 | 要件定義完了、UIデザイン、DB設計 |
| M3-M4 | コア機能開発（登録、求人CRUD、基本マッチング、チャット） |
| M5 | クローズドβテスト（東京エリア、建設業界限定） |
| M6 | MVP公開リリース |

**MVP機能スコープ：**
- 60秒レジストレーション
- 基本プロフィール + スキル登録
- 位置情報ベースの求人検索
- シンプルなマッチングアルゴリズム（ルールベース）
- アプリ内チャット
- 企業向け求人管理画面

### Phase 2: 成長（7〜12ヶ月）— "AIを解き放つ"

- AIスキルプロファイリング導入
- AIスコアリングマッチングエンジン
- スキルバッジシステム
- 現場動画 / 口コミ機能
- 緊急マッチング（スポット採用）
- 多言語対応（第一弾：英語、ベトナム語）
- 対象エリア拡大：関東全域 + 関西

### Phase 3: 拡張（13〜24ヶ月）— "エコシステムの構築"

- スキルクエスト（ゲーミフィケーション）
- インスタントペイ
- AIキャリアコパイロット
- 現場つなぎコミュニティ
- セーフティネット機能群
- 資格取得パートナー連携
- 全国展開
- 多言語拡張（6言語対応）

### Phase 4: 海外展開（25〜36ヶ月）— "アジアの現場をつなぐ"

- 東南アジア展開（ベトナム、インドネシア、フィリピン）
- 国際人材マッチング（日本↔東南アジア）
- ブロックチェーンスキル証明のグローバル標準化
- 業界別バーティカルSaaS展開
